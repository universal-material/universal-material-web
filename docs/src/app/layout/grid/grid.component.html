Grids are used to add gutter to your content Universal Material uses the new [CSS Grid](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_grid_layout) to add spacing between columns inside a grid. Currently it has one column by default and it can be customized with `--u-grid-columns`.  Also, you can directly change the CSS property `grid-template-columns` to build a more complex grid.

Variables

Variable Default
--u-grid-columns 1
--u-layout-gutter-default --u-spacing-medium (compact), --u-spacing-large  (expanded).
--u-layout-gutter --u-layout-gutter-default
--u-layout-gutter-inline --u-layout-gutter
--u-layout-gutter-block --u-layout-gutter
