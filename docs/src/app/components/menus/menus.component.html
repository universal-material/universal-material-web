Floating surfaces that present a list of choices anchored to a trigger. A `u-menu` opens above any element you call `.toggle()`/`.show()` on. By default it anchors to its parent, so a button next to the menu just works. Use the `leading-icon` slot on `u-menu-item` to add a glyph. The classic "share" pattern below.
