Groups related buttons and standardizes their spacing and alignment. The default alignment is end — the conventional place for action rows in dialogs, forms and cards. Default alignment is end. Drop two or more buttons inside and the set will space them consistently. Set alignment to start, center or end (the default). Use stack when the buttons need to be laid out vertically (e.g. on narrow viewports). The visually-prominent action comes first. The set accepts any button-like child — u-button, u-icon-button, u-fab — and aligns them on the same baseline.
