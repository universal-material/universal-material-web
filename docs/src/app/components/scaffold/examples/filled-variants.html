filled

surface-container-highest + medium (12dp) corner.

filled, filled-low and filled-lowest use the same shape corner; only the surface tone changes. Override either via --u-scaffold-pane-filled-bg-color / --u-scaffold-pane-filled-shape-corner.

filled-low

surface-container-low — softer card chrome, used by the canonical list-detail layout.
filled-lowest

surface-container-lowest — the quietest filled tone, suitable when the main column needs to pop.

Use variant="transparent" (the default) to skip the card chrome entirely.

transparent

No background, no corner. Blends with the scaffold.
