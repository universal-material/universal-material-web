filled

surface-container-low + medium (12dp) corner. The canonical card chrome for in-flow panes.

Override the bg via --u-pane-filled-bg-color and the corner via --u-pane-filled-shape-corner. When a filled pane collapses into a sidebar drawer or fullscreen overlay it switches to surface-container-lowest automatically — override --u-pane-overlay-bg-color to change that.

transparent

No background, no corner. Blends with the scaffold.
