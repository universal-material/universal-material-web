A layout container that hosts a top app bar, the scrollable page content, an optional bottom app bar, a floating FAB and optional side panes. The scaffold publishes its internal scroll element through Lit context so descendants like the top app bar and FABs can react to the same scroll source without manual wiring. Declare `u-scaffold-pane` children with `position="start"` or `position="end"` to add side regions. Each pane picks its own `collapse-breakpoint` (`sm`, `md`, `lg`, default `lg`): above it the pane joins the scaffold grid, below it the pane leaves the flow and renders as an overlay anchored to the scaffold. `collapse-mode="sidebar"` slides in a modal drawer with a scrim, `collapse-mode="fullscreen"` takes over the whole scaffold (top app bar included). For a list-detail layout, set the start (navigation) pane to `sidebar`/`lg` and the end (detail) pane to `fullscreen`/`md` — the list and main column stay side-by-side until the viewport drops below 840px.
