A layout container that hosts a top app bar, the scrollable page content, an optional bottom app bar, a floating FAB and optional side panes. The scaffold publishes its internal scroll element through Lit context so descendants like the top app bar and FABs can react to the same scroll source without manual wiring. Declare `u-scaffold-pane` children with `position="start"` or `position="end"` to add side regions. At ≥ 1200px they sit alongside the main content. Below that, they collapse and open as either a modal sidebar (`collapse-mode="sidebar"`) or a fullscreen overlay (`collapse-mode="fullscreen"`). The `variant` attribute switches between `transparent` and a `filled` card-style surface.
