A layout container that hosts a top app bar, a flex-row body, an optional bottom app bar and a floating FAB. The body accepts plain content and any number of <u-pane> children as siblings — panes auto-arrange by DOM order, and non-pane children flex to fill the remaining space. Declare any number of <u-pane> children. The pane's mode attribute drives its behaviour: fixed (default, always visible), collapsible (in flow when open, removed otherwise), sidebar (modal drawer overlay when open) and fullscreen (full-width modal overlay when open, sits above the top app bar). Override the mode at each Material 3 breakpoint with mode-sm, mode-md, mode-lg, mode-xl — applied purely via CSS, so no matchMedia wiring is needed. Use query-context="container" to react to the scaffold's own size rather than the viewport.
