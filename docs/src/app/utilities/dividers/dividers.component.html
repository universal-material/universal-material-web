Dividers are thin lines that group content in lists or other containers Create an element with `.u-divider` to group content. The default divider class has a small margin. Use `.u-divider-no-margin` if you don't need the margin.

Variables

Variable Default
--u-divider-thickness 1px
--u-divider-opacity .4
--u-divider-color --u-current-text-color
